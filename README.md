# SuperStore_Data_Analysis
# Superstore Sales Data Analysis 📊

This repository contains a comprehensive Data Cleaning and Exploratory Data Analysis (EDA) of the Sample Superstore dataset. The goal is to identify sales trends, profit margins, and regional performance to help in data-driven decision-making.

## 🎯 Project Objectives
* Clean the raw dataset by handling missing values and correcting data types.
* Identify the most profitable product categories and sub-categories.
* Analyze regional sales distribution across the United States.
* Visualize sales trends over different shipping modes and segments.

## 🛠️ Technical Skills Applied
* **Data Cleaning:** Handled null values, removed duplicates, and fixed date formats using `pd.to_datetime`.
* **Exploratory Data Analysis (EDA):** Used `.groupby()` and `.agg()` for complex data summaries.
* **Outlier Detection:** Identified unusual sales patterns in specific regions.

## 📂 Files in this Repository
* `superstore_project.ipynb`: The complete Python source code with visualizations.
* `SampleSuperstore.csv`: The raw dataset used for analysis.
* `Cleaned_Superstore_Data.xlsx`: The final processed and cleaned data.

## 📈 Key Insights Found
1. **Top Category:** The 'Technology' category has the highest profit margin compared to Office Supplies.
2. **Regional Performance:** The West region is leading in total sales volume.
3. **Shipping Trends:** 'Standard Class' is the most preferred shipping mode by customers.
